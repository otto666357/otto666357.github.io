## Why write a blog

I have postponed starting a blog for three years. Every time I thought about it, I would talk myself out of it with thoughts like "the content isn't good enough" or "nobody will read it."

This time I finally decided, because I realized: the biggest value of a blog is not for the reader — it's **for the writer**.

The process of articulating an idea forces you to turn a vague intuition into a precise argument. That in itself is a form of mental training.

## Why Astro

Among all static site generators, Astro is the one I find most comfortable:

- **Zero JavaScript by default** — a blog doesn't need to be an SPA
- **MDX support** — you can write React components directly inside Markdown
- **View Transitions** — page navigation doesn't flash white
- **One-click deploy to GitHub Pages** — free

## Features of this blog

- Bilingual (English & Chinese), automatic redirect based on `Accept-Language`
- Light and dark themes, follow system, with manual override
- All posts support RSS
- Full-text search (provided by Pagefind)
- Comments via Giscus (powered by GitHub Discussions)
